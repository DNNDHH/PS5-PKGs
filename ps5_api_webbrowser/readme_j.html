WebBrowserDialog - API サンプル

概要

ウェブブラウザダイアログを表示するサンプルです。

スクリーンショット


描画結果

詳細

このサンプルは、WebBrowserDialogライブラリを用いてウェブブラウザダイアログを表示するプログラムです。

操作説明

コントローラ操作
アクション 挙動
○ button ひとつ前のメニューに戻る
× button モードを決定する
up button フォーカスを移動する
right button メニューを表示します
left button メニューを非表示にします
down button フォーカスを移動する

メニュー構成
Default
Custom
Callback
Callback RegExp
ImeParam
ForPredeterminedContent
UrlExtended
SetCookie


メニュー
名前解説
Defaultウェブブラウザダイアログを表示します
Customウェブブラウザダイアログをカスタムモードで表示します
Callbackウェブブラウザダイアログを表示しコールバック条件が一致した時にゲーム側へURLパラメータを返します
Callback RegExpウェブブラウザダイアログを表示しコールバック条件が部分一致した時にゲーム側へURLパラメータを返します
NpAuthCodeウェブブラウザダイアログを開いてNPAuthCodeの取得を行います
ImeParamウェブブラウザダイアログ上で利用されるIMEのオプションを変更して起動します
ForPredeterminedContentペアレンタルコントロールの確認なしでウェブブラウザダイアログを起動します(TRC条件をクリアしたサイトのみ)
UrlExtendedウェブブラウザダイアログを表示しコールバック条件にURI最大長を指定し一致した時にゲーム側へURLパラメータを返します
SetCookieウェブブラウザダイアログを表示し設定されたCookieを取得します
